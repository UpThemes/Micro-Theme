Theme Documentation for Micro, a WP Microblog Framework

Micro is a very powerful microblogging theme framework with support for Express App and Wootumblog. It uses the UpThemes Framework 2.0 and supports 4 widget areas, contextual layouts, child themes, and advanced typography via Google Web Fonts.

How to Install

Install Micro Through WordPress:

  1. Log into WordPress through the admin panel. (i.e. www.yourdomain.com/wp-admin)
  2. Go to the "Appearance" module in the left navigation and select "Themes"
  3. You should see two tabs "Manage Themes" and "Install Themes", Click on "Install Themes".
  4. Now click on "Upload" in the sub nav.
  5. Select the Micro ZIP file located on your local machine.
  6. Click Install.
  7. After your theme installs you should get a confirmation that it was successful, you will have the option to "Activate", "Preview", or "Return to Themes Page".
  8. Click "Activate".
  9. Success! Micro Theme is now active.

Install Micro Manually:

  1. Upload the Micro theme files to the themes folder located on your server. (i.e. wp-content/themes/)
  2. Once this is completed log into WordPress through the admin panel. (i.e. www.yourdomain.com/wp-admin)
  3. Go to the "Appearance" module in the left navigation and select "Themes".
  4. You should see two tabs "Manage Themes" and "Install Themes" click on "Manage Themes".
  5. In this tab you will see "Current Theme" which is the standard WordPress theme or perhaps another theme you had installed previously. Also,"Available Themes" which are the list of themes that you have uploaded to your server. You should see Micro 1.0 in the available themes section. Click "Activate".
  6. Success! Micro Theme is now active.
